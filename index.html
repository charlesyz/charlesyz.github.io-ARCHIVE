Tetris Neural Network

Over the 2018 March Break, I creating a machine learning AI in Python that plays Tetris.

The Tetris AI uses a three layer neural network with 200 input neurons (each tile of the 20x10 game grid), 24 hidden neurons, and 4 output neurons (rotate, move left, move right, do nothing). To improve, the AI uses a genetic algorithm evolution system with a roulette selection system.

Maze-Bot "Beep Boop"

Built with the help of Miasya Bulger, MazeBot, also named "Beep Boop", is a autonomous maze solving robot. Using an ultrasonic sensor to detect maze walls infront of it and Infrared Sensors to follow the maze grid, MazeBot is able to navigate and solve any custom 4x4 maze. The maze walls are made of lasercut MDF boards, and the robot uses a custom depth-first-search algorithm. Mazebot is based on the Arduino Uno Microcontroller.

Tapemeasure Android App

TapeMeasure is an android app where users can use their phone to mimic a real tape measure. The app uses the built-in accelerometer to track your phone's movement. Simply slide your phone across a smooth surface, and the app will tell you how long far you've moved!

Unfortunately, due to technology limitatues (in the iPhone 5 at the time) with the accuracy of the accelerometer and noisy data, the App was not published. The best accuracy I could achieve was about +-5%, even with calibrations and noise-reduction.

In IOS12, Apple implemented a similar measuring app called Measure. Instead of using the accelerometer, It uses the camera to produce distances in real time. Some day I hope to try my accelerometer concept using the new hardware in iPhone X+

  • Source Code is on my Github

From 2017 up untill I graduated in 2018, I was the President of Lisgar's DECA club.
